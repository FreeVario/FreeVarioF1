Setup of a Cloud 9 Setup

sudo apt-get install python-dev

Install ARM tools

sudo add-apt-repository -y ppa:terry.guo/gcc-arm-embedded
sudo apt-get update
sudo apt-get install gcc-arm-none-eabi


Install MBED cli

sudo pip install mbed-cli
sudo pip install markupsafe

cd FreeVario
run "mbed deploy" to install mbed

You can install all missing modules by running "pip install -r requirements.txt" in "/home/ubuntu/workspace/FreeVario/mbed-os"


Create build command

Name: STM32 Builder

// Create a custom Cloud9 build system - similar to the Sublime build system
// For more information see https://docs.c9.io/custom_runners.html
{
    "cmd" : ["mbed", "compile", "-t", "GCC_ARM", "-m", "NUCLEO_F103RB"],
    "env" : {},
    "info" : "Building $project_path/$file_name",
    "selector": "source.ext"
}


When building, select the FreeVario Folder
